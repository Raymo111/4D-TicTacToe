4D TicTacToe

Press r after changing the values to start a new game.

Press [ and ] to switch between 2D, 3D, 4D views of the board.

Press c to connect to multiplayer room specified by room code. If there's no room code, it switched back to local game.

Keys

  • [ ] switch camera dimension
  • 🠔 🠖 move left/right
  • 🠕 🠗 move up/down
  • w s move forward/back
  • a d move akta/ana
  • ␣ place piece
  • r restart game
  • c (dis)connect server
  • t toggle menu
